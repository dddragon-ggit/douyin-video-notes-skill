# 视频笔记助手 - 小程序聊天 UI 设计方案

## 用户体验流程

```
1. 用户在抖音/B站看到一个视频
2. 点击"复制链接"
3. 打开微信 → 下拉 → 选择"视频笔记助手"小程序
4. 粘贴链接，点发送
5. 看到"正在处理..."的动画
6. 30-60秒后，笔记以对话气泡形式出现
7. 可以继续发下一个链接，形成对话记录
```

## 关键技术点

### 1. 长任务处理

云函数有超时限制（最多 60 秒），视频处理可能超过这个时间。

方案：拆成多个云函数，用状态机串联。

```
downloadVideo (10s)
      ↓
extractAudio (5s)
      ↓
transcribeAudio (20s)  ← 最耗时
      ↓
generateNotes (10s)
      ↓
saveToDatabase (2s)
```

前端轮询获取当前步骤和进度。

### 2. Markdown 渲染

笔记是 Markdown 格式，小程序不原生支持。

方案：使用 `towxml` 或 `wemark` 等小程序 Markdown 渲染库。

### 3. 链接识别

用户可能直接粘贴抖音分享文本（包含链接和描述文字），需要正则提取 URL。

```javascript
const urlRegex = /https?:\/\/[^\s]+/;
const match = text.match(urlRegex);
```

### 4. 气泡样式

```
用户消息：右对齐，绿色/蓝色背景
机器人消息：左对齐，白色/灰色背景
系统消息：居中，小字灰色
```

### 5. 文档下载

云函数生成笔记后，同时生成文件并上传到云存储。

**支持格式**：
- `.md` — Markdown 原文，直接复制粘贴到笔记软件
- `.docx` — Word 文档，方便打印和分享

**实现流程**：

```
笔记生成完成
    ↓
云函数生成 .md 文件内容
    ↓
（可选）用 python-docx 生成 .docx
    ↓
上传到微信云存储
    ↓
返回 fileID 给前端
    ↓
前端调用 wx.cloud.downloadFile 下载
    ↓
调用 wx.shareFileMessage 分享到微信
    或调用 wx.saveFileToDisk 保存到手机
```

**前端按钮逻辑**：

```javascript
// 下载 Markdown
async function downloadMarkdown(noteId) {
    const { fileID } = await db.collection('notes').doc(noteId).get();
    const { tempFilePath } = await wx.cloud.downloadFile({ fileID: fileID.mdFileID });
    wx.shareFileMessage({ filePath: tempFilePath });  // 分享到微信
}

// 下载 Word
async function downloadWord(noteId) {
    const { fileID } = await db.collection('notes').doc(noteId).get();
    const { tempFilePath } = await wx.cloud.downloadFile({ fileID: fileID.docxFileID });
    wx.saveFileToDisk({ filePath: tempFilePath });  // 保存到手机
}
```

### 6. 历史笔记

**云数据库结构**：

```javascript
// notes 集合
{
    _id: "note_001",
    _openid: "user_xxx",          // 自动写入，区分用户
    title: "AI量化交易：为什么...",
    author: "暴躁哐哐",
    platform: "douyin",
    url: "https://v.douyin.com/xxx",
    key_points: ["...", "..."],
    detail_markdown: "# AI量化...",
    quotes: ["...", "..."],
    transcript: "原始转录文本...",
    mdFileID: "cloud://xxx.md",    // 云存储文件 ID
    docxFileID: "cloud://xxx.docx",
    created_at: "2026-05-24 10:30:00"
}
```

**查询逻辑**：

```javascript
// 按时间倒序获取历史笔记
const { data } = await db.collection('notes')
    .orderBy('created_at', 'desc')
    .limit(50)
    .get();

// 按日期分组显示
const grouped = groupByDate(data);  // { "2026-05-24": [...], "2026-05-20": [...] }
```

**历史页面功能**：
- 按日期分组展示
- 点击进入详情页
- 左滑删除
- 顶部搜索（按标题关键词）
